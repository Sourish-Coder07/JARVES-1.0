## 🚀 Features

- 🎤 Voice-controlled assistant  
- 🗣 Text-to-speech responses  
- 📅 Date and time queries  
- 🎶 YouTube music playback  
- 📚 Wikipedia summaries  
- 📝 Task manager (add, speak, notify)  
- 🌐 Google search  
- ⚙️ Open desktop apps  
- 📧 Send Gmail messages  
- 💬 WhatsApp message automation  
- 🧩 Fully customizable in `user_config.py`


## 🛠 Setup Instructions

# 1. Create virtual environment
python -m venv env_jarvis

# 2. Activate environment
.\env_jarvis\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt



## ⚙️ Configure `user_config.py`

user_name = "Your Name"
email_sender = "your_email@gmail.com"
email_receiver = "receiver_email@gmail.com"
email_password = "your_gmail_app_password"
whatsapp_number = "+91XXXXXXXXXX"
music_links = ["https://youtube.com/xyz"]
voice_id = 0  # 0 = Male, 1 = Female
speech_rate = 170


## ▶️ Run the Assistant

python main.py


## 🗣️ Example Commands

> Say these aloud after launching:

* ✅ “Hello” → Greets you
* ✅ “Say time” → Tells current time
* ✅ “New task finish homework” → Adds a task
* ✅ “Speak task” → Reads your tasks
* ✅ “Play music” → Plays YouTube link
* ✅ “Wikipedia Albert Einstein” → Summarizes Wikipedia
* ✅ “Search Google Python basics” → Searches in browser
* ✅ “Send WhatsApp” → Sends message
* ✅ “Send email” → Sends Gmail


## 📁 File Structure

main.py              → Core assistant logic  
speech.py            → Text-to-speech engine  
command_input.py     → Voice input handler  
user_config.py       → Config file (your settings)  
todo.txt             → Your personal task list  
requirements.txt     → All Python dependencies  
README.md            → Project documentation


## 🌐 Internet Usage
